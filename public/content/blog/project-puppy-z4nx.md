# Building Puppy_z4nx

Puppy is my personal portfolio website built using modern web technologies.

## Tech Stack

1. **React**: The core framework for building the UI.
2. **Tailwind CSS**: Utility-first CSS for styling.
3. **Framer Motion**: Enables smooth animations.
4. **React Router**: Handles page navigation.

## Why This Stack?

I chose this tech stack because it strikes a perfect balance between performance, developer experience, and design flexibility.
